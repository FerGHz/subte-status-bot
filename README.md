# Subte Status Bot

A simple Python bot that posts automated public transportation (subway) status updates to a subreddit using Reddit's official API via PRAW.

## Purpose

This bot provides structured, low-frequency updates about subway service status for a local community subreddit.

It helps Redditors access relevant commuting information directly within Reddit.

## Features

- Uses Reddit OAuth2 authentication
- Posts scheduled updates (e.g., hourly)
- Posts only structured text updates
- Does NOT scrape Reddit
- Does NOT collect user data
- Complies with Reddit API Terms

## Technology

- Python
- PRAW (Python Reddit API Wrapper)
